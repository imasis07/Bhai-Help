## ROASTrack — Project Handoff Prompt

### About this project
ROASTrack is an influencer ROI tracker SaaS for Indian D2C brands. 
It is a **100% frontend-only** React + Vite + TypeScript project — 
NO backend, NO Clerk, NO API keys, NO database needed.
Must run standalone: `npm install && npm run dev` from the project folder.

### Tech Stack
- React 19 + TypeScript + Vite 7
- Tailwind CSS v4
- Framer Motion (animations)
- Recharts (charts)
- Wouter (routing — NOT react-router)
- Lucide React (icons)
- @tanstack/react-query
- All data is hardcoded/mock — no real API calls

### Design Tokens (NEVER change these)
- Background: hsl(222,47%,6%)
- Card: hsl(222,47%,9%)
- Input: hsl(222,47%,12%)
- Primary (cyan): hsl(199,89%,48%)
- Border: rgba(14,165,233,0.18)
- Font: system default, weights 400/500/600/700

### Current Routes (wouter)
- `/` → Landing page (Hero, Features, Pricing sections)
- `/sign-in` → Custom login page (Google button on top, then email+password)
- `/sign-up` → Custom signup page (Google button on top, then email+password, password strength bar)
- `/verify-email` → 6-digit OTP page with paste support
- `/onboarding` → Store name input page
- `/dashboard` → Setup wizard (3 steps: Connect Store, Install Pixel, Add Influencer)
- `/connect-store` → Connect store page
- `/install-pixel` → Install pixel page
- `/add-influencer` → Add influencer page
- `/main-dashboard` → Full premium dashboard (this is the main screen)
- `/pricing` → Pricing page
